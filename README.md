### Elixir

```sh

```

Batch create:
```ssh
chapter=997 && mkdir ./$chapter && touch ./$chapter/my_solution.ex && touch ./$chapter/solution.ex && alias x="iex ./$chapter/my_solution.ex"
```
> then you can use `x` for quick debug.
